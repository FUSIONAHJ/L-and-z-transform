一个交互式网页应用，用于直观展示拉普拉斯变换（s 平面）与 Z 变换（z 平面）之间的映射关系。
通过拖拽操作和参数调整，帮助理解连续时间系统到离散时间系统的转换原理。

项目介绍 
该工具通过可视化方式呈现 s 平面与 z 平面的对应关系，让用户可以：
• 直观理解复平面映射的数学原理 
• 观察采样周期（T）对变换结果的影响 
• 掌握连续系统与离散系统的稳定性判据（s 平面左半部分对应 z 平面单位圆内部）  
适合控制理论、信号处理学习者及相关领域工程师使用，通过交互操作加深对变换关系的理解。

 核心功能 
• 实时交互映射：在 s 平面拖拽紫色标记点，z 平面会实时显示对应的粉色映射点 
• 采样周期可调：通过滑块调整 T 值（范围 0.1s-2.0s），观察映射关系变化 
• 坐标实时显示：同步展示 s 平面坐标（  ）和 z 平面坐标（实部 + 虚部） 
• 稳定性区域标记： 
◦ s 平面左半部分以浅绿色阴影标示（连续系统稳定区域） 
◦ z 平面单位圆内部以浅绿色阴影标示（离散系统稳定区域）   
• 网格辅助定位：s 平面的网格线和 z 平面的同心圆网格，便于坐标参考 
• 多设备支持：适配桌面端（鼠标操作）和移动端（触摸操作）
